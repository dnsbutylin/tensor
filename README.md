# Qa -Tests UI

## Зависимости

* Python 3.12+
* poetry 1.3.2+
* корень_проекта/browser_tools/ваш_хромдрайвер.exe

## Установка зависимостей для тестов

```bash
make setup
```

## Запуск тестов

```bash
make all-tests
```

## Запуск форматеров

```bash
make format
```

## Запуск линтеров

```bash
make lint
```
